# House-Prices-Random-Forest-Regressor

House Price Prediction - Kaggle Competition
This repository contains a machine learning solution for the Kaggle Housing Prices Competition. The project demonstrates the application of regression techniques to predict real estate prices.

## Project Description
The objective of this project is to predict the sales prices of residential properties in Ames, Iowa. The solution involves data processing and the implementation of a Random Forest model to minimize the Mean Absolute Error (MAE).

## Model Performance
**Validation MAE**: 21,857[cite: 2]
- **Kaggle Public Score**: 21217.91640

- ## Model and Features
- Model Architecture: Random Forest Regressor
- Features Used: LotArea, YearBuilt, 1stFlrSF, 2ndFlrSF, FullBath, BedroomAbvGr, TotRmsAbvGrd

- ## Repository Structure
- notebook.ipynb: Includes data loading, model training, and prediction generation.
- submission.csv: Final predictions formatted for Kaggle submission.
